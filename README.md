# MERN Dev testing

## Installation:

Open your terminal

### Clone the repository

```
https://github.com/SirAnderss/devt-test.git

cd devt-test
```

### Configure env file

```
cp .env.example .env
```

And paste mongodb password in the .env file

### Install modules

```
npm install
```

### Build frontend bundle files

```
npm run build
```

### Run local server

```
npm start
```

It's done!!!

> you need the mongodb password for the app to work
